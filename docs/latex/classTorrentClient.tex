\doxysubsection{Detailed Description}
Manages torrent downloading and uploading, coordinating between different components. 

This class integrates the \doxylink{classFileManager}{File\+Manager}, \doxylink{classPieceManager}{Piece\+Manager}, \doxylink{classTorrentParser}{Torrent\+Parser}, \doxylink{classTrackerClient}{Tracker\+Client}, and Peer\+Connections to manage the overall process of handling a torrent, from parsing the file to managing downloads and uploads.
