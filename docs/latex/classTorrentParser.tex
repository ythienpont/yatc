\doxysubsection{Detailed Description}
Provides functionality to parse torrent files. 

The \doxylink{classTorrentParser}{Torrent\+Parser} class reads torrent files, decodes their contents, and extracts key information such as tracker URLs, file details, and piece hashes necessary for torrent downloading.
